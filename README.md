# Úklid schodů

Webová aplikace pro správu rozvrhu úklidu společných prostor v bytovém domě.

## Funkce

- Automatické generování týdenního rozvrhu úklidu na celý rok
- Správa seznamu sousedů (přidání, odebrání, změna pořadí)
- Přiřazení sousedů k jednotlivým týdnům
- Zvýraznění aktuálního týdne
- Možnost úpravy adresy domu
- Tisk rozvrhu pro vyvěšení
- Automatické ukládání do localStorage

## Použití

1. Přidejte sousedy pomocí tlačítka "+ Přidat souseda"
2. Přetažením změňte pořadí sousedů
3. Kliknutím na jméno v rozvrhu můžete upravit přiřazení
4. Pro tisk použijte Ctrl+P nebo funkci tisku v prohlížeči

## Technologie

- HTML, CSS, JavaScript (vanilla)
- Žádné závislosti na externích knihovnách
- Hostováno na Azure Static Web Apps
